# 数据集格式转换

竞技比赛常用的数据集格式变换。

## 变A：原始格式 → arena格式

```
原始：{"text": "内容", "answer": "类别中文名"}
目标：{"sampleId": "1-0001", "question": "内容"}
```

## 标签映射（中文→编号）

```
涉警 → 1, 邪教迷信 → 2, 社会稳定 → 3, 民族 → 4,
危安 → 5, 警务秘密 → 6, 恐怖极端 → 7
```

## 答案独立保存

```python
# answers.jsonl: {"sampleId": "1-0001", "label": "1"}
# dataset.jsonl: {"sampleId": "1-0001", "question": "内容"}
```

## sampleId按标签分组编号

按 label 前缀分组，每组从 0001 开始独立编号：
`1-0001, 1-0002, ... 2-0001, 2-0002, ...`

## 生成小测试集

从大数据集里每类随机抽 N 条，保持标签分布：
`1-01 ~ 1-10, 2-01 ~ 2-10, ... 7-01 ~ 7-10`
